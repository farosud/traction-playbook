# Traction Playbook

**19 actionable growth channel playbooks for startups.**

Based on the book *Traction* by Gabriel Weinberg & Justin Mares. Each channel is a complete, executable playbook — not generic advice. Every playbook includes service integrations, scoring frameworks, outreach templates, and measurement systems.

---

## How to Use This

1. **Pick a project** you want to grow
2. **Scan all 19 channels** to find the best fit
3. **Run the Bullseye analysis** — test 3 channels, double down on what works, kill the rest
4. **Execute** using the step-by-step playbooks

The key insight from Traction: *"Most startups don't fail because they can't build a product. They fail because they can't get traction."*

## The Bullseye Framework

Don't try all 19 channels at once. Use the Bullseye:

**Inner Ring (test now):** Pick the 3 channels with highest fit score. Run cheap, fast experiments.

**Promising Ring:** 3-5 channels that look good but need more data.

**Long Shot Ring:** Channels that might work later but don't fit your current stage.

For each Inner Ring channel, define:
- **Test:** specific experiment to run
- **Budget:** time and/or money needed
- **Metric:** what to measure
- **Timeline:** how long to run
- **Success:** what "working" looks like
- **Kill:** when to abandon
