# Search Engine Optimization (SEO) — Traction Channel 7

## Service Layer
- Serper: SERP analysis, keyword competition, featured snippet research
- Exa.ai: find content ranking for target keywords, identify content gaps
- Firecrawl: scrape competitor pages for structure/content analysis
- Jina Reader: quick single-page content extraction (free fallback)

## On Invocation

### 1. Gather Context
- Product/site URL (or planned URL)
- Current domain authority / age (new site vs established)
- Target audience (who should find you via search?)
- Existing content? (blog, docs, landing pages)
- Competitor sites (who ranks for terms you want?)

### 2. Keyword Research

#### Keyword Categories:
Generate keywords in each bucket:

A. PROBLEM KEYWORDS (people searching for the problem)
   "[how to] [problem your product solves]"
   "[problem] for [audience]"
   Format: 20-30 keywords, estimated volume + difficulty

B. SOLUTION KEYWORDS (people searching for the solution type)
   "[product category] tool/software/app"
   "best [category] for [use case]"
   Format: 15-20 keywords

C. COMPARISON KEYWORDS (people comparing options)
   "[competitor] alternative"
   "[competitor A] vs [competitor B]"
   "[product category] comparison [year]"
   Format: 10-15 keywords

D. LONG-TAIL GOLD (low competition, high intent)
   Very specific queries with clear purchase/signup intent
   "[specific use case] [specific constraint] [solution]"
   Format: 20-30 keywords

E. QUESTION KEYWORDS (informational, builds authority)
   "what is [concept related to product]"
   "how does [category] work"
   "why [problem] happens"
   Format: 15-20 keywords

#### Keyword Prioritization Matrix:
Score each keyword:
  SEARCH VOLUME: [high/med/low]
  COMPETITION: [high/med/low]
  INTENT: [informational/commercial/transactional]
  CONVERSION POTENTIAL: [1-10]
  CAN WE RANK? (realistic): [yes/maybe/unlikely]
  PRIORITY: [now/soon/later]

### 3. Content Strategy

For each priority keyword cluster:
  TARGET KEYWORD:
  CONTENT TYPE: [landing page / blog post / comparison page / guide / tool page]
  SEARCH INTENT: [what the searcher actually wants]
  OUTLINE: [H1, H2s, key sections]
  WORD COUNT TARGET: [based on competing content]
  UNIQUE ANGLE: [what makes your content better than what ranks now]
  INTERNAL LINKS: [what other pages on your site to link to/from]
  CTA: [what action should the reader take]

Content types to consider:
- Pillar pages (comprehensive guides, 3000+ words)
- Blog posts (specific topics, 1000-2000 words)
- Comparison pages (you vs competitors)
- Use-case pages (one per specific use case)
- Programmatic pages (template × data = hundreds of pages)
- Tool pages (free tools that rank and convert)
- Glossary/wiki pages (define terms in your space)

### 4. Programmatic SEO
If applicable, design a programmatic SEO strategy:

TEMPLATE: [what's the page template]
DATA: [what varies per page]
URL STRUCTURE: [site.com/[category]/[specific]]
EXAMPLE PAGES: [5 examples of what these pages look like]
SCALE: [how many pages this creates]
UNIQUE VALUE: [why these aren't thin/duplicate content]

### 5. Technical SEO Checklist
Evaluate and fix:
- [ ] Page speed (< 3s load time, Core Web Vitals)
- [ ] Mobile-friendly (responsive design)
- [ ] SSL/HTTPS
- [ ] XML sitemap submitted to Google Search Console
- [ ] robots.txt properly configured
- [ ] Canonical tags on all pages
- [ ] Meta titles (< 60 chars, keyword included)
- [ ] Meta descriptions (< 160 chars, compelling)
- [ ] Header hierarchy (single H1, logical H2-H6)
- [ ] Image alt text
- [ ] Internal linking structure
- [ ] No broken links (404s)
- [ ] Schema markup (Organization, Product, FAQ, HowTo)
- [ ] Open Graph tags (for social sharing)

### 6. Link Building Strategy
Realistic approaches for early-stage:
- Guest posts on relevant blogs (see /targeting-blogs)
- HARO/Connectively responses (journalist backlinks)
- Product directories and listings
- Open source projects linking back to you
- Creating linkable assets (tools, data, guides)
- Broken link building (find broken links on relevant sites, offer yours)

### 7. Quick Wins
Pages/keywords you could rank for within 30 days:
- Very low competition long-tail keywords
- Your brand name (should rank #1)
- Specific feature names or unique terms
- Local SEO terms if applicable

### 8. 90-Day SEO Plan
Month 1: Technical SEO fixes + 4 cornerstone content pieces
Month 2: 8 blog posts targeting long-tail + link building starts
Month 3: Programmatic pages (if applicable) + first ranking assessments

### 9. Measurement
- Keyword rankings (track weekly)
- Organic traffic growth
- Pages indexed
- Backlinks acquired
- Click-through rates from SERPs
- Conversion rate from organic traffic
- Page-level performance (which content converts)
